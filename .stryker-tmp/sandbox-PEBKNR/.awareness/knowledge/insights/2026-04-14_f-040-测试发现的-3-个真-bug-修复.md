# F-040 测试发现的 3 个真 bug + 修复

补单元测试时发现：(1) pickSuggestions(null) 因默认参数失效 crash，修复用 meta = meta || {}；(2) renderAuthPending 把 verification_uri 直接进 href 没校验协议，javascript: URL 是真 XSS 漏洞，修复强制 /^https?://i 否则 about:blank；(3) i18n EN/ZH 不对齐（chip 三个 key 漏译）。教训：核心路径必须有 attack-string XSS 测试 + null/undefined 边界测试。
